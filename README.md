# API42 [![Gem Version](https://badge.fury.io/rb/API42.svg)](http://badge.fury.io/rb/API42)

A simple ruby wrapper for the 42 API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'API42'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install API42
